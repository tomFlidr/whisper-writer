# WhisperWriter – Skills Index

This file is the entry point for all skill documents.
Load the relevant skill file before performing the corresponding action.

| Action | Skill file | When to load |
|---|---|---|
| Writing or reviewing any code | `docs/skills/coding-standards.md` | Always – before writing or reviewing code |
| Creating a git commit | `docs/skills/commit.md` | Before every commit |
| Creating a release | `docs/skills/release.md` | Before bumping version or pushing a tag |
| Refactoring existing code | `docs/skills/refactoring.md` | Before any refactoring task |
| Implementing a new feature | `docs/skills/new-feature.md` | Before starting new functionality |
| Investigating a bug or crash | `docs/skills/debugging.md` | When diagnosing unexpected behaviour |

> Skills may reference each other. For example, finishing a new feature requires
> following the commit skill afterwards.
